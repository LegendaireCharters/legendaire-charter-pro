# Legendaire Charter Pro — HTML v6 Lite

Plain HTML/CSS/JavaScript PWA for GitHub Pages.

## Changes in this version
- Removed quote preset buttons.
- Removed aircraft recommendation logic and labels.
- Removed per-aircraft quote cost comparison from aircraft cards.
- Added manual aircraft-selection note.
- Added Internal view / Client view toggle for quote output.
- Rebuilt the WA SVG map with a more recognisable WA outline, state border and animated route aircraft.
- Kept flat landing fee default at $200 and landing fee per passenger default at $0.

## Upload
Upload the contents of this folder to your existing GitHub Pages repository, replacing the old files.

If the Home Screen app shows an old version, open the web URL in Safari and refresh once, then reopen the Home Screen app.
