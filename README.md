# Legendaire Charter Pro — HTML/PWA v5

Plain HTML/CSS/JavaScript app for GitHub Pages. No build step required.

## Upload from iPhone
1. Unzip this folder in Files.
2. Open your GitHub repo in Safari.
3. Upload/replace all files and folders from this package.
4. Commit changes.
5. Open your GitHub Pages URL in Safari and refresh.
6. Add to Home Screen again if iOS keeps showing an old cached version.

## Included
- Professional Legendaire-style quote calculator
- Custom SVG WA route map with animated aircraft
- Searchable airport/mine/station strip database
- Aircraft recommendation and comparison cards
- Positioning sector support
- Flat landing fee default $200
- Per-passenger landing fee default $0
- One-page print/save quote sheet
- Recent quotes saved locally on device
- Offline PWA support

## Important
Airstrip details are indicative only. Confirm runway, permission, fuel, NOTAMs/ERSA/operator approvals before quoting or operating.
